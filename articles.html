Les défis de Google Apps Script

Lors de mon stage, j'ai rencontré plusieurs difficultés en utilisant Google Apps Script pour automatiser et améliorer les processus sur Google Spreadsheet. L'interface de Google Apps Script n'est pas très pratique et manque de fonctionnalités de base que l'on trouve dans les éditeurs de code modernes, comme la détection d'erreurs. C'est comme écrire du code dans un simple fichier texte, ce qui peut rendre le développement plus fastidieux.

De plus, Google Apps Script est assez limité et bridé. Certains scripts plus complexes ne fonctionnent pas de manière cohérente, même lorsque les mêmes conditions d'utilisation sont reproduites. Cela peut être frustrant et nécessite souvent des ajustements supplémentaires pour s'assurer que les scripts fonctionnent correctement dans tous les cas.
